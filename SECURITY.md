# Security Policy

## Supported Versions

Security fixes are provided for the latest stable minor release line.

## Preferred Reporting Channel

Use GitHub Private Vulnerability Reporting when available for this repository.

## Alternative Reporting Channel

If private reporting is unavailable, email: security@avenra.dev

## Reporting a Vulnerability

Include:

- Affected version
- Reproduction steps
- Potential impact
- Suggested mitigation (if available)

## Response Targets

- Initial acknowledgement: within 3 business days
- Triage decision: within 7 business days
- Ongoing status updates: at least every 14 days until resolution

## Disclosure Policy

- Do not open public issues for undisclosed vulnerabilities.
- Coordinate disclosure timing with maintainers.
- Credit reporters in release notes unless anonymity is requested.

## Out of Scope

- Vulnerabilities in unsupported dependency versions
- General hardening advice without a specific reproducible issue
